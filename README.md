# Amazon Sales Revenue Analytics Dashboard

## Project Overview

This project analyzes Amazon sales data to uncover revenue trends, customer behavior, regional performance, and product category insights. The workflow includes data cleaning using Python, business analysis using PostgreSQL, and interactive dashboard development in Power BI.

## Business Problem

E-commerce companies generate large volumes of sales data, but raw data often contains inconsistencies, duplicates, and quality issues. The objective of this project is to transform raw sales records into actionable business insights that support decision-making.

## Project Workflow

### 1. Data Cleaning & Preprocessing (Python)

Performed:

- Missing value analysis
- Duplicate record removal
- Date format conversion
- Feature engineering:
  - Year
  - Month
  - Quarter
- Revenue validation
- Outlier detection
- Exploratory Data Analysis (EDA)

### 2. SQL Business Analysis (PostgreSQL)

Conducted business-focused analysis including:

- Total Revenue
- Total Orders
- Revenue by Product Category
- Quantity Sold by Category
- Revenue by Region
- Revenue by Payment Method
- Monthly Revenue Trend
- Quarterly Revenue Trend
- Customer Rating Analysis
- Discount Impact Analysis
- Top 10 Revenue Generating Products
- Revenue Contribution Percentage

### 3. Dashboard Development (Power BI)

Built an interactive multi-page dashboard featuring:

#### Executive Overview
- Total Revenue KPI
- Total Orders KPI
- Monthly Revenue Trend
- Revenue by Region
- Top Categories by Revenue
- Revenue by Payment Method

#### Sales Performance Analysis
- Category Performance
- Revenue Distribution
- Discount Impact Analysis
- Product-Level Performance

#### Customer & Regional Insights
- Regional Revenue Contribution
- Customer Satisfaction Analysis
- Regional Rating Comparison
- Revenue vs Customer Metrics

## Tools & Technologies

- Identified top-performing product categories driving revenue.

- Analyzed customer ratings across regions.

- Improved data quality through cleaning and validation.
- Generated actionable business insights using SQL.
- Developed an executive-ready Power BI dashboard for decision-making.
